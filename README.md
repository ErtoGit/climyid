# CLIMYID
CLIMYID is a simple cli tool for your server.

## Features
- [x] Update Linux machine (CentOS, Debian, Ubuntu)
- [ ] Install LEMP stack (Nginx, MySQL, PHP 7x) + Let's Encrypt
- [ ] Install LAMP stack (Apache, MySQL, PHP 7x) + Let's Encrypt
